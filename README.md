---

## 要求

- PHP >= 7.2
- Laravel >=5.8

---

## 安装

```bash
composer require martialbe/laravel-baidu-aip
```

---

## 开始使用

1. 创建配置文件

```bash
php artisan vendor:publish --provider="Martialbe\LaravelBaiduAip\ServiceProvider"
```
